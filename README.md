该项目是一个简单的脚手架(template)
包含以下技术栈：
1. Springboot 4.0.5
2. Eureka Server 5.0.1
3. Spring Cloud 2025.1.1
4. spring-cloud-starter-gateway-server-webmvc
5. spring-cloud-starter-netflix-eureka-client
6. spring-cloud-starter-openfeign
7. spring-cloud-starter-alibaba-sentinel

JDK 版本：21

包含：服务注册与管理、网关、服务发现、熔断机制
spring-cloud-starter-alibaba-sentinel 只是客户端，需要搭配https://github.com/alibaba/Sentinel/releases的jar客户端一起使用。
